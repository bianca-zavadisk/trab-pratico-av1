Função Objetivo (Minimização de Penalidades)

O algoritmo atribui uma pontuação $S(c)$ para cada cor candidata $c$, baseada no mínimo das distâncias e preferências em relação às cores $p$ já presentes na paleta $P$:

$$ S(c) = w_{dist} \cdot \min_{p \in P} (\Delta E(c, p)) + w_{pref} \cdot \min_{p \in P} (\text{Pref}(c, p)) $$

Onde $\Delta E$ é a distância Euclidiana e a Preferência é aproximada por:

$$ \text{Pref}(c, p) = 47.61 \cdot |\Delta L| - 46.42 \cdot \Delta \text{Hue} $$

Código em Execução (JavaScript)

function calculateScores(cand, palette, wDist, wPref) {
    let minDist = Infinity, minPref = Infinity;

    for (const p of palette) {
        // 1. Distância Euclidiana em LAB
        const dl = cand.lab.l - p.lab.l;
        const da = cand.lab.a - p.lab.a;
        const db = cand.lab.b - p.lab.b;
        const dist = Math.sqrt(dl*dl + da*da + db*db);

        // 2. Preferência de Par (Lightness & Hue)
        const dhue = Math.abs(cand.hue - p.hue);
        const pref = (47.61 * Math.abs(dl)) - (46.42 * dhue);

        if (dist < minDist) minDist = dist;
        if (pref < minPref) minPref = pref;
    }

    const finalScore = (wDist * minDist) + (wPref * minPref);
    return { finalScore, minDist, minPref };
}
